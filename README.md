# EvacMEI Data + Android Improvement Draft

Este repositorio contiene datos de reglas de evacuación y una propuesta de mejoras para una app Android de emergencia minera.

## Qué se agregó

- Documento de mejoras UX/UI y arquitectura para Android Studio: `docs/android_mejoras.md`.
- Ejemplo de pantalla en Jetpack Compose con animación y botón de actualización: `samples/android/EmergenciaMineScreen.kt`.
- Motor mínimo de resolución `ubicación + foco -> destino`: `samples/android/RouteEngine.kt`.

## Datos existentes

`data_pack.zip` incluye reglas y catálogo:

- `MEI_evac_rules.json`
- `MEI_catalogo_destinos.json`
